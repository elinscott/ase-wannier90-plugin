##############
 Installation
##############

The most recent release can be installed from `PyPI
<https://pypi.org/project/ase_wannier90_plugin>`_ with uv:

.. code-block:: console

    $ uv pip install ase_wannier90_plugin

or with pip:

.. code-block:: console

    $ python3 -m pip install ase_wannier90_plugin

*********************
 Installing from git
*********************

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/elinscott/ase-wannier90-plugin.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/elinscott/ase-wannier90-plugin.git

****************************
 Installing for development
****************************

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/elinscott/ase-wannier90-plugin.git
    $ cd ase-wannier90-plugin
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
